# 💱 Currency Exchange Converter

A dynamic, real-time currency converter built with **vanilla JavaScript**, **Express.js**, and public REST APIs. Converts between 150+ global currencies with live exchange rates and country flags.


## ✨ Features
- **Live exchange rates** from [currency-api](https://github.com/fawazahmed0/currency-api)
- **150+ currencies** with country flags
- **Dynamic dropdowns** populated from local API
- **Responsive UI** with form validation
- **Full-stack architecture**: 
  - Frontend: HTML/CSS/JavaScript
  - Backend: Node.js + Express (local server)

## 🛠️ Technologies Used
- **Frontend**: 
  - HTML5, CSS3, JavaScript (ES6+)
  - DOM manipulation, `fetch()`, async/await
- **Backend**: 
  - Node.js, Express.js
  - CORS-enabled REST API (`/countrycode`)
- **APIs**: 
  - [currency-api](https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api) (exchange rates)
  - [flagcdn](https://flagsapi.com) (country flags)
